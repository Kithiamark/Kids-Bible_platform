# Exercise 04: Data and Objects

## Goal

Get used to reading and shaping structured data.

## Practice Data

```python
student = {
    "id": 1,
    "display_name": "Mercy",
    "current_level": 3,
    "total_points": 240,
}
```

## Drills

1. Read the student's name and level.
2. Increase the points by 10.
3. Write a function that returns a short summary string.
4. Create a second student and compare their levels.
