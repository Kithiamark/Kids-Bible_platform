# Exercise 03: Naming and Small Functions

## Goal

Learn how good names and smaller functions make code easier to read.

## Rename These

Turn these into better names:
- `data`
- `thing`
- `handleStuff`
- `value2`
- `temp`
- `x`

## Split This Idea

Take this job:
- "grade quiz submission and update student points"

Split it into smaller function ideas, for example:
- validate attempt
- grade answers
- calculate score
- update student points
- build response

## Reflection

Ask:
- would someone understand the name without extra explanation
- does each function idea have one main job
- is any name too vague
