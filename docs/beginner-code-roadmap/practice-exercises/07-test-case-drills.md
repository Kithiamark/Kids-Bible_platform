# Exercise 07: Test Case Drills

## Goal

Practice spotting good test cases before writing code.

## Drill 1: Passing Score

Function:
- `is_passing_score(score, threshold)`

Write at least:
- one happy path
- one failure path
- one edge case

## Drill 2: Quiz Submission

Think about these test cases:
- valid quiz submit
- attempt not found
- already completed attempt
- zero max points
- wrong quiz id

For each one, write:
- setup
- action
- expected result

## Drill 3: Route Protection

For a parent-only route, list test cases for:
- parent user
- admin user
- teacher user
- student user
- unauthenticated user
