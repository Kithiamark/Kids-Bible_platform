# Exercise 01: Variables and Functions

## Goal

Get comfortable with:
- storing values
- passing values into functions
- returning a result

## Drills

1. Create a variable called `student_name` and store a name in it.
2. Create a variable called `score` and store a number in it.
3. Write a function called `is_passing_score(score, passing_score)`.
4. Make it return `True` when the score is high enough.
5. Test it with 3 different values.
