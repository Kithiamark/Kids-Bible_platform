# Checklist: Am I Guessing?

- Can I clearly say what is broken?
- Do I know what I expected to happen?
- Do I have evidence from logs, output, or a test?
- Am I changing one thing at a time?
- Did I read the error message fully?
- Do I know which file or layer I am investigating?
- If I am stuck, did I shrink the problem into a smaller piece?
