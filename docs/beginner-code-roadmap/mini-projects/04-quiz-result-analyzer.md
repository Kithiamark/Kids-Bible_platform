# Mini Project 04: Quiz Result Analyzer

## Goal

Create a small program that takes quiz results and produces a summary.

## Suggested Inputs

```python
results = [
    {"student": "Mary", "score": 85},
    {"student": "John", "score": 65},
    {"student": "Ruth", "score": 90},
]
```

## Suggested Output

- highest score
- lowest score
- average score
- list of passing students
- list of failing students

## What This Teaches

- reading lists of objects
- looping
- conditions
- clean helper functions
- deciding what deserves a test
