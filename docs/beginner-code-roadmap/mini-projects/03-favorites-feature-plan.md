# Mini Project 03: Favorites Feature Plan

## Goal

Plan a small feature without writing messy code.

Feature idea:
- let a student mark a lesson as favorite

## What You Should Decide

- what data needs to exist
- what backend route is needed
- what schema is needed
- what service logic is needed
- what frontend page or component should trigger it
- how you would show saved favorites

## Placement Guide

Try to decide:
- model: favorite record or relation
- schema: favorite create/response
- API: favorite endpoint
- service: add/remove favorite logic
- frontend API helper: request wrapper
- page/component: button and list

## Test Ideas

- favorite a lesson successfully
- favorite same lesson twice
- remove favorite
- student tries to favorite a lesson outside their allowed group
