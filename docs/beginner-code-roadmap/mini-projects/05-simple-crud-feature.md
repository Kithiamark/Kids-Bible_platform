# Mini Project 05: Simple CRUD Feature

## Goal

Plan and build a very small create/read/update/delete style feature.

Good example:
- notes
- tags
- achievements
- saved messages

## Beginner Plan

1. choose one tiny entity
2. list its fields
3. decide what model would store it
4. decide which schemas are needed
5. decide which routes are needed
6. decide if service logic is needed
7. decide how frontend pages/components interact with it

## Keep It Small

Do not build a full system.

Build only:
- create
- list
- update one field
- delete

## Test Ideas

- create valid item
- reject bad input
- list existing items
- update existing item
- reject missing item
- delete existing item
