# Mini Project 01: Score Calculator

## Goal

Build a tiny function-based project that:
- takes quiz scores
- calculates total and average
- decides pass/fail

## Why This Project Matters

It teaches:
- functions
- lists
- conditions
- small clean logic
- simple test cases

## Suggested Pieces

- `calculate_total(scores)`
- `calculate_average(scores)`
- `is_passing_score(score, threshold)`
- `build_summary(student_name, scores, threshold)`

## Test Ideas

- empty scores list
- all passing
- mixed scores
- score exactly equal to threshold
